GitEvents

Event Management meets Issue Ops

What is GitEvents?

GitHub Issues + Your Event = GitEvents. Successful event management with the tools you know.

GitHub Issues

GitEvents uses a GitHub Issues to create, track and manage your Events.

GitHub Actions

GitEvents uses a single GitHub Action with various event triggers.

Automation

Automatically propogate events out to Social Networks (Facebook, Twitter, ...) and Event Management sites and keep people informed (Tweets, Status Updates, Email).
